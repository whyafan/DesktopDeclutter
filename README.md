# Desktop Declutter

A macOS app for decluttering your desktop with a Tinder-like swipe interface. Review files quickly by swiping left to bin, right to keep, or stack for later review.

## Features

- **Swipe Interface**: Intuitive card-based interface for quick file review
- **Quick Look Preview**: Preview files without leaving the app
- **Stack for Later**: Mark files for batch review before deletion
- **Immediate or Delayed Binning**: Choose to trash immediately or collect for review
- **Keyboard Shortcuts**: 
  - `←` (Left Arrow): Bin current file
  - `→` (Right Arrow): Keep current file
  - `S`: Stack current file
  - `Spacebar`: Preview file in Quick Look
- **Progress Tracking**: See your progress and statistics as you declutter

## Building

```bash
swift build
```

## Running

```bash
swift run
```

## Requirements

- macOS 13.0 or later
- Swift 5.9 or later
